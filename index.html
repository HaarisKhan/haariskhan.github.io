Welcome to My Website!

A little about me...

Hey! I'm Haaris, a junior studying Computer Science at UC Berkeley. I intend to use this website for two purposes:

1. A CS learning blog. Arguably the primary reason I'm making this website, I want this page to be a place where I can walk through data structure and algorithm proofs and examples. As a student, I find that many of the neat data structures or algorithm techniques lack examples or are quite hand-wavy. I want this blog to highlight interesting / misunderstood data structures and algorithms and walkthrough the understanding of them with my own twist!

2. A project portfolio. I want this site to also be a history of the projects I've completed, no matter how mundane or astounding. Source code for non-class projects will be provided; if interested in the class projects, please contact me!
